# Enterprise AI Automation with Microsoft 365 Copilot Cowork

## Overall Estimated Duration: 8 Hours

## Overview

In this hands-on workshop, you will explore Copilot Cowork, the agentic workspace inside Microsoft 365 Copilot, and learn how it automates multi-step enterprise work across Outlook, Calendar, Teams, OneDrive, SharePoint, and Excel. Across eight progressive labs, you will move from getting oriented with Cowork to building a daily workday assistant, optimizing your calendar, coordinating the full meeting lifecycle, governing enterprise files, building custom Cowork skills, assembling a personal work dashboard, and finally building and publishing your own Copilot Cowork plugin.

>**Disclaimer:** Copilot Cowork is AI-powered, so generated outputs may differ across users and sessions. The results you see in this lab may not exactly match the examples shown here. Follow the workflow and expected outcome in the guide rather than expecting identical content.

## Objectives

In this lab, you will perform the following:

- Get oriented with Copilot Cowork and run your first agentic tasks across Outlook, OneDrive, and Teams
- Build a daily workday assistant that briefs you from your calendar, email, and Teams activity
- Detect and resolve calendar conflicts and manage an executive-style calendar with protected focus time
- Automate the full meeting lifecycle, from AI-generated preparation to cross-app content collection and follow-up
- Classify, rename, and govern enterprise files across OneDrive and SharePoint with human-in-the-loop approvals
- Design and build custom Cowork skills using the conversational Skill Builder
- Build a self-refreshing personal work dashboard delivered to your inbox every morning
- Build and publish a custom Copilot Cowork plugin connected to an external data source

## Pre-requisites

Participants should have:

- Access to Microsoft 365 Copilot with Copilot Cowork enabled
- A Microsoft account with permissions to use Outlook, Calendar, Teams, OneDrive, SharePoint, and Excel
- Basic familiarity with Microsoft 365 applications
- A modern web browser such as Microsoft Edge or Chrome
- Lab environment credentials, if provided by the facilitator
